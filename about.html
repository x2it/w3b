Chapter 01

项目概览

知行工作室（w3b.pub）是一个以"知行合一"为核心理念的个人品牌展示网站。网站通过粒子动画、时间线叙事、交互式平衡器和嵌入式内容模块，将中国传统哲学思想与现代 Web 技术、AI 理念相融合，呈现"理会"（思考与洞察）、"践行"（行动与创造）与"AI 赋能"（数字演进）的三位一体。

1.1 设计理念

网站围绕四条核心线索展开：

  • 理会 · 洞察本质 — 通过粒子动画可视化思维的四种状态（有序、发散、专注、AI 神经网络），呼应朱熹"穷理"的思想
  • 践行 · 当下行动 — 通过时间线串联从《尚书》到王阳明再到当下的知行思想脉络
  • 知行合一 · 动态平衡 — 通过交互式天平让用户直观感受知与行的平衡关系
  • AI 赋能 · 数字演进 — AI 是知与行的数字映射：数据为知，决策为行。神经网络中每一层都是"知"，每一次前向传播都是"行"

Chapter 03

页面模块

3.1 粒子动画区（理会 · 洞察本质）

基于 Canvas 的粒子系统，支持四种思维状态切换，其中 AI 模式将知行合一理念与神经网络可视化融合：

  • 有序思维 — 粒子排列为规则网格，象征逻辑与秩序
  • 发散联想 — 粒子自由运动并相互连线，象征创造性思维
  • 专注凝聚 — 粒子向中心聚拢，象征深度思考与聚焦
  • AI 思维 — 粒子形成 3 层神经网络结构（输入层 → 隐藏层 → 输出层），层间连线模拟突触连接，信号光点沿连线流动并带拖尾效果，节点发光模拟神经元激活。象征 AI 作为知与行的数字映射

80-120 个粒子，requestAnimationFrame 驱动，支持鼠标/触摸交互。颜色跟随主题变量动态变化。

3.2 时间线（践行 · 当下行动）

纵向时间线展示知行思想的历史脉络：

  • 《尚书》 — "非知之艰，行之惟艰。" 中国思想史上对知行关系最早的著名论断
  • 朱熹 — "只有两件事：理会，践行。" 强调穷理与力行的先后次序
  • 王阳明 — "知是行的主意，行是知的功夫；知是行之始，行是知之成。" 在心学体系中实现知与行的本体统一
  • 当下 — 每一个项目、每一次选择，都是古老思想在当下的实践

3.3 平衡器（知行合一 · 动态平衡）

交互式天平可视化，用户通过两个滑块控制"理的深度"和"行的力度"，天平横梁实时倾斜反映平衡状态。

3.4 思考笔记 & 项目作品

通过 iframe 嵌入第三方内容，展示最新的思考文章和项目作品。

4.2 神经网络背景

页面背景添加 SVG pattern 平铺的神经网络节点 + 连线纹理，opacity 仅 0.03，极微妙不抢主体内容，fixed 定位跟随页面滚动。

4.3 AI 赋能交互

卡片 hover 时添加基于 --accent-rgb 变量的微妙光晕效果（rgba(var(--accent-rgb), 0.03)），让交互带有 AI 驱动的数字质感。

Chapter 05

主题系统

网站支持 8 种主题（默认、暗色、蓝色、绿色、紫色、橙色、青色、系统跟随），通过 CSS 变量实现全局切换。主题偏好通过 localStorage 持久化保存，刷新页面后自动恢复。

主题切换触发方式：① 手动点击右上角 🎨 按钮
② 系统暗色模式自动适配（当选择"系统"主题时）

Chapter 07

部署与维护

  1. 代码推送到 GitHub 仓库 x2it/w3b 的 main 分支
  2. GitHub Pages 自动构建并部署到 w3b.pub
  3. CDN 缓存约 5-10 分钟刷新，通过 style.css?v=xxx 参数强制刷新
维护注意事项：iframe 高度需根据第三方页面内容变化定期调整；第三方 iframe 内容由外部维护，无法控制其样式；每次修改 style.css 后必须更新 ?v= 版本参数。
